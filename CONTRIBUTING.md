# Contributing

本專案歡迎 privacy-conscious 的資料整理、文件修正、圖表改善與方法論討論。

## 可以貢獻

- 修正文件錯字或不清楚的說明。
- 改善資料邊界與 privacy review 流程。
- 增加 aggregate-only demo data。
- 增加不涉及個人追蹤的圖表。
- 回報資料錯誤、圖表誤導或隱私風險。

## 不接受

- 原始 tracking data。
- 個人層級敏感時間序列。
- 真名、住址、學校、工作地點、私人帳號。
- 中之人推測或未公開身份推測。
- 即時監控資料。
- 可造成 doxxing 或騷擾的內容。

## Pull Request Checklist

提交前請確認：

- 沒有新增禁止欄位。
- 沒有新增未聚合的敏感資料。
- 若新增資料，已附上 source URL 或來源說明。
- 若新增資料，已附上 last verified。
- 若新增圖表，不會反推出個人作息、身份或敏感狀態。

## Correction and Removal

若貢獻內容被指出有安全、隱私或誤解風險，維護者可先移除或降精度處理，再討論後續修正。
