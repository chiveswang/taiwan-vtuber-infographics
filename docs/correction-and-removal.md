# Correction and Removal Requests

本專案接受 correction、removal 與 opt-out request。

## Correction

適用於資料錯誤、來源過期、分類錯誤或圖表標示不清。

建議提供：

- 受影響的檔案或圖表。
- 錯誤描述。
- 建議修正。
- 可公開驗證的來源。

## Removal / Opt-Out

適用於資料雖然公開，但放入本專案後可能造成安全、隱私或誤解風險。

維護者可採取：

- 移除資料列。
- 移除圖表。
- 降低資料粒度。
- 合併分類。
- 改寫文字說明。

## 處理原則

安全與隱私優先於資料完整性。若資料有爭議，先暫停公開，再評估是否重新發布。
