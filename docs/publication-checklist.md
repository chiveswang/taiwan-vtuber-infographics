# Publication Checklist

公開任何資料或圖表前，先檢查：

- [ ] 沒有原始 tracking data。
- [ ] 沒有真名、住址、學校、工作地點、私人帳號或私人聯絡方式。
- [ ] 沒有中之人推測或未公開身份推測。
- [ ] 沒有即時追蹤或個人層級敏感時間序列。
- [ ] 敏感指標已聚合到 month、quarter、year 或足夠大的群組。
- [ ] 樣本太小的分類已合併或不發布。
- [ ] 每份資料都有 source URL 或來源說明。
- [ ] 每份資料都有 last verified。
- [ ] 圖表不會鼓勵排名、騷擾或個人比較。
- [ ] correction / removal request 路徑清楚。
