## 建議描述

This repository is a privacy-conscious open-source data visualization project focused on aggregate trends in the Taiwan VTuber ecosystem.

The project does not publish raw tracking data, private identity information, doxxing-related material, or sensitive individual-level time series. Public outputs are limited to aggregate charts, public source links, methodology notes, and correction/removal request processes.

The goal is to make a small, transparent OSS project that demonstrates responsible data handling, reproducible derived datasets, and public-interest visualization without exposing personal information.

## 避免用語

- complete tracking database
- real-time monitoring
- personal profile database
- identity research
- 中之人分析
- 個人作息分析
