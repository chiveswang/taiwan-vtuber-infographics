# Privacy Policy

本專案是 privacy-conscious public data visualization project。公開內容應以聚合、衍生、可驗證、低風險為原則。

## Scope

本專案可能整理公開可見的 VTuber 生態資訊，並將其轉換為 aggregate trends、charts、summaries 與 methodology notes。

本專案不公開原始 tracking data，也不建立個人層級追蹤資料庫。

## 禁止收錄或公開的資訊

- 真名、住址、電話、私人 email。
- 學校、工作地點、住處、常出沒地點。
- 中之人推測、未公開身份推測、私人關係推測。
- 私人帳號、私人照片、私人直播紀錄。
- 即時追蹤、即時活躍狀態。
- 可推測個人作息的細粒度時間序列。
- doxxing 相關內容。

## 可公開資訊原則

可公開資料應符合以下條件：

- 資料來源為公開頁面或官方公開連結。
- 欄位必要、低風險，且與圖表目的直接相關。
- 附上 source URL 或來源說明。
- 附上 last verified date。
- 若資料可能造成個人風險，應先聚合或移除。

## Aggregation Principle

可能敏感的活動頻率、成長趨勢、活動時間、合作網絡或內容分類，應以月、季、年或足夠大的群組發布。樣本太小時，應合併分類或不發布。

## Correction Request

若資料不正確、過期或引用方式造成誤解，可以提出 correction request。請提供：

- 受影響的檔案、圖表或資料列。
- 具體問題。
- 建議修正方式。
- 可公開查證的來源，若有。

## Removal / Opt-Out Request

即使資料來自公開來源，如果公開於本專案會造成安全、隱私或誤解風險，也可以提出 removal 或 opt-out request。

維護者可視情況移除、模糊化、聚合或改寫相關資料。安全與隱私優先於資料完整性。

## Maintainer Response

維護者應以最小揭露、善意處理與可回溯修正為原則。對於有爭議的資料，應先下架或降精度，再評估是否重新發布。
