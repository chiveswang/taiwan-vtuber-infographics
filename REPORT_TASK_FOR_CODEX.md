# 工單：年度產業報告（自動敘事，新分頁 `s_report`）

> 把各分頁的關鍵數字串成一篇可讀的「台 V 產業回顧」。樣板句 + 即時填數，附「複製全文」方便分享。
> 只改 `build_full_dashboard.py`，全部用既有內嵌資料（ACT / COH / growth / streaming），不需新資料。

## 接點
1. TABS 加 `["s_report","產業報告",buildReport]`。
2. body 加 `<section id="s_report">`：年度選擇器 + `<div id="reportBody">` + 「複製全文」鈕 + 口徑/警語摺疊。
3. `function buildReport(){…}`：依選年填樣板、組 HTML。

## 年度範圍
- 年度選擇器（如 2023/2024/2025/最新狀態）。預設**最新已結算年**。
- YoY 與趨勢句以「該年最後已結算季 vs 前一年同季」計算（沿用既有 settled 過濾）。

## 章節與樣板（每節 2–4 句，{ } 即時填）
1. **產業規模**：追蹤頻道 {tracked}、近期活躍 {active}（活躍率 {rate}%，YoY {±pp}）；YouTube 訂閱總量 {yt_subs}（YoY {±%}）、Twitch 追隨 {tw_fol}。 ← ACT
2. **進出場動態**：出道高峰 {peak_q} 單季 {peak_debuts} 人，{該年} 出道 {debuts}／畢業 {grads}，累積淨活躍 {cum}；{擴張/成熟} 期。 ← COH
3. **內容趨勢**：熱門影片以 {最大類別} 為主，短影音佔 {shorts%}（YoY {±pp}）；直播內容主幹為 {直播最大類別}。 ← ACT buckets
4. **平台版圖**：YouTube 活躍佔 {yt%}、Twitch {tw%}；雙棲 {dual} 頻道。 ← ACT（crossplatform 若已就緒則補主場/傾斜趨勢）
5. **集中度**：前 10 大佔全體訂閱 {top10%}（{較前年升/降}），訂閱中位數 {median}（{變化}）→ {長尾變厚/頭部集中}。 ← ACT
6. **世代組成**：國籍以 {TW/HK/MY…前三} 為主；團體 : 個人 = {g}:{i}。 ← COH
7. **生命週期**：典型台 V 出道 {x} 個月達訂閱中位 {median@x}；前段(p75) {p75@x}。 ← growth.lifecycle
8. **開台生態**：開台尖峰 {peak_hour} 時（台灣）、{peak_weekday}；{季節性一句，如年末/暑假高峰}。 ← streaming.industry

## 呈現
- 每節：小標 + 敘事段落（粗體標關鍵數字）。整體像一頁可讀報告，不是圖表牆（圖在各自分頁）。
- **「複製全文」**：把純文字版（去 HTML）複製到剪貼簿，方便貼到社群/文件。
- **口徑/警語摺疊**（預設收合）：資料來源、倖存者偏差、proxy、作息覆蓋度、無 SC/同接/互動率。

## 驗收
- [ ] 切年度，數字與趨勢句隨之更新；最新年預設正確。
- [ ] 8 節敘事完整、數字與其他分頁一致（抽查 top10%、出道高峰等）。
- [ ] 「複製全文」可複製純文字。
- [ ] crossplatform 未就緒時，平台節以 ACT 既有數字撐住、不報錯。
- [ ] 全 client-side、內嵌，無 console error。

## 警語（摺疊區固定）
- 數字為自動從資料生成；倖存者偏差、proxy(黏著/事件)、作息覆蓋度(取樣星期) 同其他分頁。
- 本資料源不含 SC 金流、即時同接、互動率、剪輯頻道。
